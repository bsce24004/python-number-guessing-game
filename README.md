# 🎯 Number Guessing Game

A simple Python terminal game where the player guesses a randomly generated number.

## Features

- Random number between 1 and 100
- Too High / Too Low hints
- Very Close / Close hints
- Best Score tracking
- Play Again option
- Input validation
- Attempt counter

## Technologies Used

- Python

## How to Run

1. Open terminal
2. Run:

```bash
python number_guess.py
```
